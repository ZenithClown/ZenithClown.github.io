   .. rubric:: Getting Started

   The code is publically available at `GitHub
   gists <https://gist.github.com/ZenithClown>`__ which is a simple
   platform for sharing *code snippets* with the community. To use the
   code, simply clone the code like:

   .. code:: shell

      git clone https://gist.github.com/ZenithClown/e4a0183d9124b3b78cce5a5dc8607323.git pandas_
      export PYTHONPATH="${PYTHONPATH}:pandas_"

   Done, you can now easily import the function with *python*
   notebooks/code-files like:

   .. code:: python

      import pandas_ as pd_ # kept same convention
